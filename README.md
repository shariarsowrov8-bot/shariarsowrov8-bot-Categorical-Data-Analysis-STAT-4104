## 🛠️ Methodological Architecture
This research portfolio handles advanced categorical and count framework evaluations across seven critical data problems:

1. **Problem 1:** Multivariable Logistic Regression modeling of respiratory lung disease.
2. **Problem 2:** Experimental evaluation of exercise program efficacy via One-Way ANOVA with Tukey HSD post-hoc evaluation.
3. **Problem 4:** Fisher's Exact Test using the Freeman-Halton matrix extension for sparse clinical setups.
4. **Problem 5:** Maximum Likelihood Graduate Admission tracking via Logistic GLM frameworks.
5. **Problem 6:** Count data assessment via Poisson GLMs using canonical Log-link functions.
6. **Problem 7 & 11:** Modeling overdispersed, long-tailed counts via Zero-Truncated and Negative Binomial (NB2) structural expansions.
7. **Problem 8 & 9:** Separation of structural and sampling zero processes via Zero-Inflated Poisson (ZIP) and Zero-Inflated Negative Binomial (ZINB) mixtures.
